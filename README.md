# Android - Comparacion de Arquitecturas de Datos

Proyecto Android para explorar y comparar diferentes enfoques de manejo de datos en aplicaciones moviles. Sirve como referencia para entender la evolucion desde implementaciones sencillas hacia arquitecturas mas robustas.

## Enfoques comparados
- Acceso a datos directamente desde Activity (sin arquitectura)
- Separacion con Repository Pattern
- Arquitectura MVVM con ViewModel y LiveData

## Por que importa
Cada enfoque tiene implicaciones en:
- **Testabilidad:** facilidad de escribir pruebas unitarias
- **Mantenibilidad:** claridad y organizacion del codigo
- **Escalabilidad:** facilidad de agregar nuevas funcionalidades
